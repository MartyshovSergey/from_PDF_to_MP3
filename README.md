# from_PDF_to_MP3

Программа для создания MP3 из файла PDF.

### Технологии:

* Python 3.7
* pdfplumber
* gtts
* art

### Запуск:

Установить и запустить виртуальное откружениие:

```python
python3 -m venv venv
source venv/bin/activate
```

Установить необходимы баблиотеки:

```python
pip install --upgrade pip
pip install pdfplumber gtts art
```
